# Page Size

Page size determines the number of rows displayed per page.  
This helps manage large data sets by breaking them into smaller, more manageable chunks.

## How Page Size Works

Use the `pageSize` prop to define how many rows are shown on each page.

## Code Example

Below is an example of how to implement page size in the DataTable component:

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const PageSizeDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", firstName: "John", lastname: "Doe" },
      { id: "2", firstName: "Jane", lastname: "Smith" },
      { id: "3", firstName: "Alice", lastname: "Johnson" },
      { id: "4", firstName: "Bob", lastname: "Brown" }
    ]}
    columnSettings={[
      { title: "ID", column: "id" },
      { title: "First Name", column: "firstName" },
      { title: "Last Name", column: "lastname" }
    ]}
    pageSize={2}
  />
);
```
