# Column Sorting

Column sorting allows you to order data by clicking on a column header. This feature helps users quickly organize and analyze table data.

## How Column Sorting Works

When enabled via the `enableColumnSorting` prop, you can control the sort order for individual columns using the `sort` attribute in the `columnSettings` array. The `sort` attribute accepts the following values:

- **`'asc'`** – The column is sorted in ascending order.
- **`'desc'`** – The column is sorted in descending order.
- **`false`** – Sorting is disabled for the column.

This enables dynamic sorting based on user interaction.

## Code Example

Below is an example of how to implement column sorting in the DataTable component:

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const ColumnSortingDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", firstName: "John", lastname: "Doe" }
    ]}
    columnSettings={[
      // This column is sorted in ascending order.
      { title: "ID", column: "id", sort: 'asc' },
      // This column is sorted in descending order.
      { title: "First Name", column: "firstName", sort: 'desc' },
      // Sorting is disabled for this column.
      { title: "Last Name", column: "lastname", sort: false }
    ]}
    enableColumnSorting
  />
);

```
