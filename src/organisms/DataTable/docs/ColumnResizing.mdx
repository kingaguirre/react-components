# Column Resizing

Column resizing allows users to adjust the width of table columns, providing flexibility in viewing data.

## How Column Resizing Works

When enabled via the `enableColumnResizing` prop, you define column dimensions by specifying `width`, `minWidth`, and `maxWidth` in the `columnSettings` array. These settings control the default width as well as the minimum and maximum allowed widths for each column.

## Code Example

Below is an example of how to implement column resizing in the DataTable component:

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const ColumnResizingDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", firstName: "John", lastname: "Doe" }
    ]}
    columnSettings={[
      // This column has a default width of 100, with a minimum of 80 and a maximum of 200.
      { title: "ID", column: "id", width: 100, minWidth: 80, maxWidth: 200 },
      // This column has a default width of 150, with a minimum of 100 and a maximum of 250.
      { title: "First Name", column: "firstName", width: 150, minWidth: 100, maxWidth: 250 },
      // This column also has a default width of 150, with a minimum of 100 and a maximum of 250.
      { title: "Last Name", column: "lastname", width: 150, minWidth: 100, maxWidth: 250 }
    ]}
    enableColumnResizing
  />
);

```
