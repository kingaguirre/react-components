# Selected Rows

Selected rows represent the currently chosen rows from the table, which can be used for further actions.

## How Selected Rows Work

Use the `selectedRows` prop to pass an array of selected row IDs.  
This allows the table to render the selection state for those rows.

## Code Example

Below is an example of how to implement selected rows in the DataTable component:

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const SelectedRowsDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", firstName: "John", lastname: "Doe" },
      { id: "2", firstName: "Jane", lastname: "Smith" },
      { id: "3", firstName: "Alice", lastname: "Johnson" }
    ]}
    columnSettings={[
      { title: "ID", column: "id" },
      { title: "First Name", column: "firstName" },
      { title: "Last Name", column: "lastname" }
    ]}
    selectedRows={["1", "3"]}
  />
);
```
