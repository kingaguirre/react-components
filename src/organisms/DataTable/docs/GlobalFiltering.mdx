# Global Filtering

Global filtering allows users to search and filter data across the entire table, making it easier to locate specific information.

## How Global Filtering Works

When enabled via the `enableGlobalFiltering` prop, the table displays a search input that filters rows across all columns.

## Code Example

Below is an example of how to implement global filtering in the DataTable component:

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const GlobalFilteringDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", firstName: "John", lastname: "Doe" }
    ]}
    columnSettings={[
      { title: "ID", column: "id" },
      { title: "First Name", column: "firstName" },
      { title: "Last Name", column: "lastname" }
    ]}
    enableGlobalFiltering
  />
);
```
