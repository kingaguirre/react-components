# Row Selection

Row selection enables users to select individual rows for further actions, such as editing or deletion.

## How Row Selection Works

When enabled via the `enableRowSelection` prop, the table provides checkboxes or similar UI elements for selecting rows.

## Code Example

Below is an example of how to implement row selection in the DataTable component:

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const RowSelectionDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", firstName: "John", lastname: "Doe" }
    ]}
    columnSettings={[
      { title: "ID", column: "id" },
      { title: "First Name", column: "firstName" },
      { title: "Last Name", column: "lastname" }
    ]}
    enableRowSelection
  />
);
```
