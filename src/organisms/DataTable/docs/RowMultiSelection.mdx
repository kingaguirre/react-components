# Multi Row Selection

Multi row selection enables users to select more than one row at a time.  
This is useful for bulk actions such as deletion or updates.

## How Multi Row Selection Works

When enabled via the `enableMultiRowSelection` prop, users can select multiple rows simultaneously.  
The selected rows are stored in the `selectedRows` array prop.

## Code Example

Below is an example of how to implement multi row selection in the DataTable component:

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const MultiRowSelectionDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", firstName: "John", lastname: "Doe" },
      { id: "2", firstName: "Jane", lastname: "Smith" },
      { id: "3", firstName: "Alice", lastname: "Johnson" }
    ]}
    columnSettings={[
      { title: "ID", column: "id" },
      { title: "First Name", column: "firstName" },
      { title: "Last Name", column: "lastname" }
    ]}
    enableRowSelection
    enableMultiRowSelection
    selectedRows={["1", "3"]}
  />
);
```
