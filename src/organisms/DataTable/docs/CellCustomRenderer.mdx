# Custom Cell Renderer

The `cell` property in `columnSettings` allows rendering a fully customized JSX output for each cell.  
This is useful when you need advanced rendering like formatting, icons, buttons, or conditional content.

## How It Works

Pass a function to the `cell` property that returns a JSX element or null.

## Code Example

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const CustomCellRendererDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", status: "active" },
      { id: "2", status: "inactive" }
    ]}
    columnSettings={[
      {
        title: "Status",
        column: "status",
        cell: ({ rowValue }) => (
          <span style={{ color: rowValue === "active" ? "green" : "red" }}>
            {rowValue}
          </span>
        )
      }
    ]}
  />
);
```

## Use Cases

- Render badges, tags, or icons inside cells.
- Format raw values (dates, currency, etc.).
- Provide inline buttons, toggles, or indicators.
