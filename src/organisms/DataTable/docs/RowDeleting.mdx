# Row Deleting

Row deleting allows users to remove rows from the table, simplifying data management.

## How Row Deleting Works

When enabled via the `enableRowDeleting` prop, users can remove individual rows from the data source.

### Partial Row Deletion

You can optionally use the `partialRowDeletionID` prop to enable *partial* deletion instead of permanent removal.

- When provided, deleted rows will not be removed from the data entirely.
- Instead, the row will be marked with a new key (`partialRowDeletionID`) set to `true`.
- This is useful when you want to visually remove rows but retain them in the backend or for later processing.

## Code Example

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const RowDeletingDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", firstName: "John", lastname: "Doe" },
      { id: "2", firstName: "Jane", lastname: "Smith" }
    ]}
    columnSettings={[
      { title: "ID", column: "id" },
      { title: "First Name", column: "firstName" },
      { title: "Last Name", column: "lastname" }
    ]}
    enableRowDeleting
    partialRowDeletionID="markedDeleted"
  />
);
```
