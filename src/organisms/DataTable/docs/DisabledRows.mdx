# Disabled Rows

Disabled rows are those that are not interactive; they do not respond to click events and typically have a different visual style.

## How Disabled Rows Work

Specify the `disabledRows` prop as an array of row IDs that should be disabled.

## Code Example

Below is an example of how to implement disabled rows in the DataTable component:

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const DisabledRowsDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", firstName: "John", lastname: "Doe" },
      { id: "2", firstName: "Jane", lastname: "Smith" },
      { id: "3", firstName: "Alice", lastname: "Johnson" }
    ]}
    columnSettings={[
      { title: "ID", column: "id" },
      { title: "First Name", column: "firstName" },
      { title: "Last Name", column: "lastname" }
    ]}
    disabledRows={["2"]}
  />
);
```
