# Header Right Buttons

The `headerRightButtons` prop allows you to add custom action buttons to the right side of the DataTable header.  
These buttons appear before the default "Add" button and are useful for secondary actions like exporting, filtering, or bulk operations.

## How Header Right Buttons Work
The interface allows you to configure the button-like element with the following properties:

- **`text`**: *(string)* – The label or text displayed on the element.
- **`color`**: *(ColorType, optional)* – Specifies the color theme applied to the component.
- **`variant`**: *("default" | "outlined" | "link", optional)* – Determines the visual style of the component.
- **`disabled`**: *(boolean, optional)* – If set to true, disables interactions and visually indicates the disabled state.
- **`className`**: *(string, optional)* – Allows custom styling by applying additional CSS class names.
- **`icon`**: *(string, optional)* – Defines an icon to display alongside the text.
- **`onClick`**: *(function)* – Callback function triggered when the component is clicked.

Pass an array of button configuration objects to the `headerRightButtons` prop.  
Each object defines the button’s text, appearance, icon, and behavior.

## Code Example

```tsx
import React from 'react';
import { DataTable } from 'react-components-lib';

export const HeaderRightButtonsDemo = () => (
  <DataTable
    dataSource={[
      { id: "1", name: "Alpha" },
      { id: "2", name: "Beta" }
    ]}
    columnSettings={[
      { title: "ID", column: "id" },
      { title: "Name", column: "name" }
    ]}
    headerRightButtons={[
      {
        text: "Export",
        color: "primary",
        variant: "outlined",
        icon: "download",
        onClick: () => console.log("Export clicked"),
      },
      {
        text: "Filter",
        variant: "link",
        icon: "filter",
        onClick: () => console.log("Filter clicked"),
      },
    ]}
  />
);
```
